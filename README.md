# Dietly Project Documentation

## Overview
Dietly is an innovative application designed to help users create and manage their dietary plans effectively. It provides a range of features, including meal tracking, nutritional advice, and customizable meal planning.

## Features
- **User Registration**: Allows users to create individualized accounts to track their dietary habits.
- **Meal Tracking**: Enables users to log their meals and monitor their nutritional intake.
- **Custom Meal Plans**: Users can generate meal plans tailored to their dietary preferences and goals.
- **Analytics Dashboard**: Offers insights into user habits and nutrition trends over time.

## Installation
To install Dietly, follow these steps:
1. Clone the repository: `git clone https://github.com/patrykbrodziak/dietly.git`
2. Navigate to the project directory: `cd dietly`
3. Install dependencies: `npm install`
4. Start the application: `npm start`

## Usage
Once installed, open the application in your browser and follow the on-screen instructions to set up your profile and start tracking your meals.
